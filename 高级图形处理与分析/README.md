## 授课老师

白天

## 课程介绍

课程内容包括图像处理、图像分析、图像理解相关知识。

使用教材《数字图像处理（第四版）》

## 分数安排

实验50%

期末50%

## 实验

一共五个实验

- 实验1-灰度变换
- 实验2- 直方图均衡
- 实验3- 空域滤波
- 实验4- 图像去噪
- 实验5-频域滤波及综合应用实验

## 考试

只有期末考试，开卷。

一定要带课本、PPT和笔记。

题型包括概念题、计算题和分析题

## 复习方法

上课好好听课，课后复习教材，考前结合教材把PPT看一遍。

参考复习重点：

- 直方图
- m邻接
- 图像混叠，莫尔效应
- 傅里叶变换（傅里叶公式，谱和相角）
- 振铃效应
- 每一种滤波（概念，滤波过程，滤波器，陷波滤波，高/低通滤波）
- 逆滤波，退化函数
- 彩色图像（伪彩色，计算图像大小）
- 编码冗余，LZW编码，比特平面编码
- 开运算，闭运算，腐蚀和膨胀
- 坎尼Canny（非极大抑制）
