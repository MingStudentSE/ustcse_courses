# 信息论与编码

## 授课老师

吴桂兴

##课程介绍

上课压力较小，老师主要采用英文PPT上课，每节课会将一些科学家的小故事

上课的时候会布置与期末考试相关联的作业，当堂测验和评讲

## 实验

老师在临近期末的时候会按小组安排一篇全英论文，要求做PPT的presentation

## 复习方法

上课的PPT主要对应期末考试的选择填空部分，当堂测验主要对应大题部分

考察的内容主要有

- 条件概率，熵，信息量
- Lempel-Ziv编码与解码
- 哈夫曼树的构建与编码
- 朴素贝叶斯
- 汉明码
- CRC
- 生成矩阵和校验矩阵的构建
- **卷积码**（占50%，这道题很难）
