## 课程介绍

**因为华保健老师2021不开信安课，因此该文档内容为分享信安入门资料**

华老师使用参考教材《计算机安全导论：深度实践》进行授课，基本会把这本书讲完

这本教材的作者是一位华人大牛，在十几年的教学生涯中，精心设计了30多个动手实验，终于在2017年完成了这本计算机安全教材。

本教材分为三大主题：软件安全、Web安全和网络安全。

作者从2002年就开始开发统一的信安教学环境，将项目命名为SEED（取SEcurity EDucation两个单词的前两个字母）。该项目非常成功，得到了美国国家科学基金会（NSF）的多次资助。因此本教材上的所有实验，只需要下载SEED Ubuntu系统，即可进行实践。所有的实验配套环境都在该系统上配置好了，不需要为了某漏洞换各种版本的Ubuntu系统。
